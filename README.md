Выполненное тестовое задание можно запустить через docker командой:
**Обратите внимание: после того как поднимутся контейнеры, подождите 30-60 секунд**.
```
docker compose build --no-cache && docker compose up -d
 ```
Дальеше можно увидеть работу по ссылке:
 ```
http://127.0.0.1:8000/admin/products
 ```
